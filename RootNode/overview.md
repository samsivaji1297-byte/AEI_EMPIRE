# **1. Base Needs Layer (Deepest Layer)**  
This is the *source code* of human behaviour.  
Everything else is built on top of these.

The 8 base needs:

- **Safety** — “I don’t want to lose what I have.”  
- **Belonging** — “I want to be accepted.”  
- **Competence** — “I want to feel capable.”  
- **Recognition** — “I want to be seen.”  
- **Autonomy** — “I want control over my life.”  
- **Progress** — “I want to feel like I’m moving forward.”  
- **Clarity** — “I want to understand what’s happening.”  
- **Identity** — “I want to know who I am.”

These are **infinite demand generators**.  
Every human problem is a distortion of one of these.

# **2. Root Nodes (Emotional Engines)**  
Root nodes are **emotional truths** that arise from base needs.

Examples:

- “I’m tired of doing more and getting nothing back.”

- “I can’t stay consistent.”  
- “I’m tired of repeating the same cycles.”

Root nodes are **not topics**.

They power everything.
